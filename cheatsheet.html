Rate, Speed & Distance
The foundation of motion problems

Core Triangle

d = r × t
d = distance  |  r = rate (speed)  |  t = time
Rearrange: r = d / t  |  t = d / r

Average Speed

Avg = Total Distance ÷ Total Time
⚠️ NOT the arithmetic mean of two speeds!
For equal-distance legs at speeds a, b:
Avg = 2ab / (a + b) (harmonic mean)

Opposite Directions

Gap Rate = v₁ + v₂
Two objects moving apart → distances add.
Time to be D apart: t = D / (v₁ + v₂)

Chase / Same Direction

Closing Rate = v_fast − v_slow
Faster object catches up → use speed difference.
Time to catch up: t = gap / (v_fast − v_slow)

Wind / Current

With wind: v + w  |  Against: v − w
v = speed in still air/water, w = wind/current speed.
Use d = rt for each leg separately.

Train Passing

d = L_train + L_object
Passing a platform: d = train length + platform length.
Overtaking another train: use relative speed.

⚠️ Common Mistakes

  • Averaging speeds directly — (60+40)/2 ≠ average speed. Use total dist ÷ total time.
  • Unit mismatch — Convert minutes ↔ hours before plugging into d = rt.
  • Forgetting train length — The train must clear the platform entirely.

Systems of Linear Equations
Two equations, two unknowns

Substitution Method

Solve one eq for x → Substitute
1. Isolate one variable: x = …
2. Plug into the other equation.
3. Solve → back-substitute.

Elimination Method

Add/Subtract to cancel a variable
Multiply equations so one variable has matching (or opposite) coefficients, then add or subtract.

Number of Solutions

a₁/a₂ ≠ b₁/b₂ → One solution
One solution: lines intersect.
No solution: same slope, different intercept (parallel).
Infinite: same line (proportional equations).

Word Problem Setup

Define variables → Write 2 equations
Typical setups: total count + total value,
e.g., a + c = 200 and 12a + 8c = 2080.

💡 Pro Tips

  • Quick check: Always substitute your answer back into BOTH equations.
  • SAT shortcut: Sometimes they ask for x + y or x − y directly — try adding/subtracting equations before solving individually.

Slope & Linear Equations
Lines on the coordinate plane

Slope Formula

m = (y₂ − y₁) / (x₂ − x₁)
Rise over run. Positive = uphill, negative = downhill.

Slope-Intercept Form

y = mx + b
m = slope, b = y-intercept (where line crosses y-axis).

Standard Form

Ax + By = C
Slope = −A/B. y-int = C/B. x-int = C/A.

Point-Slope Form

y − y₁ = m(x − x₁)
Use when you know slope + one point. Rearrange to y = mx + b.

Direct & Inverse Variation
Proportional relationships

Direct Variation

y = kx  (y/x = k)
As x increases, y increases proportionally.
Graph: straight line through origin.

Inverse Variation

y = k/x  (xy = k)
As x increases, y decreases.
Graph: hyperbola in quadrants I & III (if k > 0).

Joint Variation

z = kxy
z varies directly as both x and y.

Combined Variation

z = kx / y
z varies directly as x and inversely as y.

Inverse Square

y = k / x²
Common in physics: light intensity, gravity.
Double the distance → ¼ the intensity.

How to Solve

Step 1 → find k  |  Step 2 → use k
1. Plug in given values to find k.
2. Use k with new values to find the unknown.

Ratio & Work Rate
比例 · 工作效率 · 折扣 · 濃度

Ratio to Value 比值

a : b → a = bk, b = bk
Set a = mk, b = nk. Use conditions to find k.
Chain ratios: match the shared term via LCM.

Work Rate 工作效率

Rate = 1 / Time
Combined: 1/t₁ + 1/t₂ = 1/T
A does job in 6h, B in 4h → together: 1/6+1/4 = 5/12 → 12/5 = 2.4h

Discount 折扣

Final = Original × d₁ × d₂
打八折 = ×0.8, 打九折 = ×0.9
先八折再九折 = 0.8 × 0.9 = 0.72 (七二折)

Concentration 濃度

C = solute / total
Adding pure solute → solute changes, water stays.
Adding water → water changes, solute stays.

Age Problems 年齡

Now + n = Future age
Everyone ages the same amount.
Set up ratio now → write ratio after n years.

Profit 成本利潤

Profit = Selling − Cost
定價 = 成本 × markup
售價 = 定價 × discount
利潤率 = (售 − 成) / 成 × 100%

💡 Quick Tricks

  • Chain ratio 連比: a:b = 2:3, b:c = 4:5 → LCM of b = 12 → a:b:c = 8:12:15
  • Workers × Days = constant (inverse proportion for identical workers)
  • Meeting problems 相遇: combined rate = sum of speeds; gap closes at sum rate
